한국사매치

한국사능력검정시험(한능검) 자가진단 학습 도구

한국사매치란?

한국사매치는 한능검을 준비하는 학습자를 위한 사실 매칭 기반 자가진단 도구입니다. 매번 똑같은 5지 선다 풀이 대신, 실제 기출 디테일을 시대·인물·사건 키워드 카드와 매칭하면서 자주 놓치는 키워드를 자동으로 찾아드립니다.

왜 만들었나

한능검 학습의 본질은 "어떤 사실이 어느 키워드에 속하는지" 판별하는 능력. 그런데 시중 도구들은 단순 객관식 반복이라 사용자가 직접 "내가 어디서 약한지" 분석해야 했습니다.

한국사매치는 풀이 기록을 자동 집계해서:

  • 자주 틀리는 키워드를 [취약 키워드] 탭에 모아 보여줍니다
  • 시대별 학습 정도(정답률)를 한눈에 시각화합니다
  • 약점 키워드가 등장하는 다른 출제 예시를 학습 힌트로 활용합니다

학습 흐름

1
시대 선택 선사·고조선부터 현대까지 11개 시대 중 하나
2
사실 카드 제시 정답 키워드가 가려진 사실문이 노출됨 (예: "**가 시행되었다")
3
키워드 매칭 관련 키워드 카드를 탭해서 정답 제출
4
즉시 피드백 정답·오답 결과와 함께 원문 노출. 길게 누르면 학습 힌트
5
자동 약점 분석 세션 끝나면 정답률 + 자주 놓친 키워드가 자동 누적

데이터

한국사능력검정시험 공식 기출문제를 시대·키워드 단위로 분류하고 매칭 가능한 형태로 정제했습니다. 신규 회차는 시험 종료 후 1~2주 안에 추가됩니다. 잘못된 매칭이나 오류를 발견하시면 앱 내 [문제가 이상해요] 버튼으로 신고해주세요.

자주 묻는 질문

무료인가요?
네, 광고도 없습니다. 회원가입 없이 즉시 시작.
회원가입 필요한가요?
필요 없습니다. 풀이 기록은 이 기기 브라우저에만 저장됩니다.
기본 / 심화 난이도 차이는?
설정에서 선택. 기본 = 평이한 디테일(난이도 ≤3)만, 심화 = 전체 디테일.
모바일에서 앱처럼 쓸 수 있나요?
네, 홈 화면에 추가하면 PWA(Progressive Web App)로 작동합니다. 안드로이드 크롬은 설치 배너가 자동 노출되고, iOS는 공유 메뉴 → 홈 화면에 추가.
새 회차 데이터는 언제 추가되나요?
한능검 시험 종료 후 1~2주 안에 추가하는 것을 목표로 합니다.
문제 신고는 어떻게?
사실 매칭에서 [정답보기(포기)] 후 노출되는 [문제가 이상해요] 버튼, 또는 설정의 [문의사항/요청 보내기] 폼으로.
