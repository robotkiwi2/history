- **2026-05-23**: 연도 모드 인터랙션 수정. 문제 카드를 **하단**으로 이동, 탭 대상도 문제 카드로 변경(배경 카드는 비탭). 오답 시 = 삽입 안 됨 + 카드 흔들림(빨강) + "오답!" 표시 후 1.5초 뒤 hint 복구, **재시도 가능**. 정답 시 = 갭에 삽입 + "정답!" + 다음 문제 버튼. 시행착오 학습 흐름.
- **2026-05-23**: 연도 모드 핵심 변경 — **정답 카드 누적**. 정답 맞춘 카드는 영구적으로 배경에 남고(=배경의 일부), 다음 문제는 1.3초 후 자동 제시. 같은 갭에 여러 카드가 들어가면 시간순으로 자동 정렬(역사 학습 목적이라 chronological order 강제). 모든 문제 풀면 "다시 시작" 버튼 노출. 결과: 문제 풀수록 타임라인이 길어지면서 자기가 배운 사건 누적 시각화.
- **2026-05-23**: 카드 데이터 모델 통합 — **모든 카드는 `{text, year}` 동일 형식**. 초기 기준 카드("500년")도 그냥 text="500년", year=500인 카드일 뿐. 시각적 구분(`.inserted` 클래스로 색상 다름)만 유지. 배경 카드 범위 확장: **기원전 2000년 ~ 2000년 (500년 단위 9개)**. 갭 8개로 증가. 카드 누적에 대비해 카드 높이 52→36px(약 70%), 폰트 15→13px로 축소. 정답 풀(pool)도 BCE/고대 사건 추가해 12개로 확장.
